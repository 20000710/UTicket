# uTicket

## About this project
uTicket is an ordering ticket web application that built with NextJS with backend ExpressJS deployed on vercel

### Clone
```Shell 
git clone https://github.com/20000710/UTicket 
```

### Package installation
```Shell 
npm i; 
```

### Testing
```Shell 
npm start; 
```
